# InstarProject
인스타그램과 유사한 SNS 어플리케이션


패스트캠퍼스의 __Android 앱 개발의 정석 with Kotlin 올인원 패키지 Online__ 강좌의 실습 예제

* 프로젝트 기간 : 2022-10-17 - 2022-10-23

# 개발 기능
* 강좌에서 제공되는 서버상과 통신
* ID, PW를 통한 로그인 기능
* 인스타그램의 기본적인 기능들

# 개발 환경
## 언어
* Kotlin 213-1.7.10

## 개발 환경 & IDE
* Windows10
* Android 12(Api 32)
* Android Studio Dolphin l 2021.3.1

# 프레임워크 및 라이브러리
* Retrofit2 라이브러리
